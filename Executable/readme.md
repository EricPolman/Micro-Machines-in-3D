Micro Machines Clone In 3D by Eric Polman (130048)
==================================================

Instructions
------------
1. Start the executable
2. Click "Start Game"
3. Race 4 laps
4. Press "P" to return to menu

Controls
--------
* Press "Space" to accelerate
* Press "A" and "D" to change direction
* Avoid mines or use them to boost yourself'
